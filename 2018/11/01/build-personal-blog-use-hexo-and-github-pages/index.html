Hexo常用命令

init

新建一个网站。如果没有设置folder，Hexo 默认在目前的文件夹建立网站。

注：folder必须为空。

new

执行new命令，生成指定名称的文章至hexo\source_posts\postName.md。

新建一篇文章。如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout参数代替。如果标题包含空格的话，请使用引号括起来。有哪些layout呢，可以到scaffolds目录下查看，这些文件名称就是layout名称。

想要给每篇文章添加分类,在hexo/scaffolds/post.md模板中添加categories

generate

生成静态文件,参数如下：

选项 描述
-d, --deploy 文件生成后立即部署网站
-w, --watch 监视文件变动

server

启动服务器。默认情况下，访问网址为： http://localhost:4000/。

deploy

部署网站。

clean

清除缓存文件db.json和已生成的静态文件public。

在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。

更多命令见官方文档
